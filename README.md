# ⚡ Algoton

`Algoton` — мониторинг крипто-фьючерсов в реальном времени на Go и React. Сервис получает стаканы через WebSocket и показывает:

- матрицу спредов с динамикой
- кросс-обменные арбитражные возможности и алерты
- базисные возможности для TON (`DeDust` vs перпеты)
- потоковые графики и обновление интерфейса через WebSocket

## Что умеет

- подключение к 12-13 источникам spot/futures/on-chain в реальном времени
- мониторинг только пары `TONUSDT` / `TON-USD(T)` (остальные тикеры убраны)
- матрица спредов в реальном времени
- дашборд с последними сигналами по паре
- автоматический расчет ценовых дисбалансов по лучшим bid/ask
- графики на lightweight-charts
- алерты по неэффективным расхождениям

## Как это работает

### Backend (Go)

- каждый источник работает в отдельной goroutine
- сбор заказов/цен через websockets
- расчет midpoint из best bid и best ask
- расчет арбитражных сигналов и трансляция клиентам через единый websocket (`/ws`)

### Фронтенд

- React + TypeScript + Vite
- живые таблицы/графики/спреды
- отображение подключений и статуса источников

## Запуск

1. Установите зависимости и соберите статический frontend:

```bash
git clone https://github.com/stepandra/Algoton.git
cd Algoton
npm run build
```

2. Запустите backend:

```bash
go run main.go
```

3. Откройте приложение: `http://localhost:8082`

## Конфиг

- `PORT` — порт сервера (по умолчанию `8082`)
- Для лимитов по алертам можно настроить порог прямо в UI (по умолчанию `0.05%`)
